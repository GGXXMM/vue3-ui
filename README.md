## vue3-ui 简介
基于Vue3框架的UI组件库，适用于PC端

## 技术栈
- Vue 3
- TypeScript
- Vite

## 项目运行
```
// 安装依赖
pnpm install

// 本地运行
pnpm run dev

// 本地文档预览
pnpm run docs:dev

// 编译打包
pnpm run build
```

## 功能组件
- [x] Button组件
   - 类型type：primary、success、info、warning、danger
   - 尺寸size：small、default、large
   - 禁用disable：true、false
